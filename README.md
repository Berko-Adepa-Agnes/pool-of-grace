# Pool of Grace

A web-based platform empowering young women in Ghana through integrated technology education, self-worth development, sustained mentorship, and career guidance.

## GitHub Repository
https://github.com/Berko-Adepa-Agnes/pool-of-grace

## Description
Pool of Grace is a full-stack web application that addresses the psychological, cultural, economic, and technical barriers preventing young women aged 16-30 in Ghana from pursuing technology careers. The platform is 100% free and starts with self-worth development before any technical training.

## Features
- 4-step onboarding where users share their story and goals
- Self-worth introduction explaining the platform approach
- 20 learning modules across self-worth, tech skills, and career development
- 5-question knowledge check quiz per module with scoring and feedback
- 3-step mentorship session scheduling with Ghanaian female mentors
- Participant and admin dashboards
- JWT-based authentication with localStorage persistence
- Role-based routing for participants, mentors, and admins

## Tech Stack
- Frontend: React.js 18.2
- Backend: Node.js 18 with Express.js 4.18
- Database: PostgreSQL 15.3
- Authentication: JWT with bcrypt password hashing
- Styling: Inline React styles with lemon green and white color scheme
